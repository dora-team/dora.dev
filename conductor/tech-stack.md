# Technology Stack

## Core Technologies
- **Static Site Generator:** Hugo - Used for the primary content structure and static page generation.
- **Frontend Framework:** Svelte 5 - Employed for building interactive, embedded single-page applications (SPAs).
- **Programming Language:** TypeScript - Used within the Svelte components for type safety and better developer experience.
- **Styling:** Sass - Used for advanced CSS preprocessing and styling across the site and Svelte components.
- **Build Tool:** Vite - The build tool and development server for the Svelte-based interactive components.

## Infrastructure & DevOps
- **Hosting:** Firebase Hosting - Provides global, secure hosting for the static site and assets.
- **CI/CD:** GitHub Actions - Automates the build, test, and deployment pipelines.

## Testing
- **E2E Testing:** Playwright - Used for end-to-end testing of capabilities and critical user flows.
