Vitesse et stabilité

Quatre métriques clés

  • Délai d'affichage des modifications : délai entre le moment où le code est validé dans un système de contrôle des versions et le moment où ce code est disponible pour les utilisateurs.
  • Fréquence de déploiement : fréquence à laquelle les modifications sont transmises à l'environnement de production.
  • Temps de restauration du service : temps nécessaire à la récupération complète après un incident ou une panne.
  • Taux d'échec des modifications : pourcentage de modifications en production ayant échoué.
